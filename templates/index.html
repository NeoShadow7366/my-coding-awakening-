What is a Generative AI model?

Generative models produce new content (text, images, etc.) based on patterns learned from training data.

Examples: GPT, Llama, Mistral. They predict the next token in a sequence to generate a coherent response.

Ollama lets you run these models entirely on your own computer — no internet required after the model is downloaded.

Welcome! Select a model in the sidebar and start chatting. If no models appear, make sure Ollama is installed and running (ollama serve).
